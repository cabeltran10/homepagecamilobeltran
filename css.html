Link to a Stylesheet

Para poder usar CSS es necesario realizar un link con el archivo que contiene las propiedades que deseamos, para esto se usa el tag link. Este tag cuenta con varios atributos: rel, type y por ultimo href.

  • rel: Relacion con el archivo, en este caso es "stylesheet"

  • type: Especifica el typo de archivo, en este caso es "text/css"

  • href: Especifica la ruta donde se encuentra el archivo, puede ser una ruta externa.

Anatomia de una regla CSS

Las reglas se deben escribir de una manera especifica, si no se sigue dicha regla simplemente no funciona.

Algunas propiedades

  • font-family: Usado para especificar que tipografia se va a usar (i.e. Georgia, Arial, ...). Es posible asignar mas de una tipografia y seprar con (,) con el fin de que si se da el caso de que una no funciona por no estar instalada, usa la siguiente.

  • color: Usado para cambiar el color de la letra, linea o demas. Se puede usar diferentes maneras para definirlo, alguos ya existen en CSS y se pueden llamar por su nombre pero es mejor el uso de Hexadecimal (#RRGGBB) o de RGB (rgb(128,0,128)). De ambas maneras es posible usarse.

  • .class: Usado para referirse a alguna clase, con el fin de poder asignarle valores personalizados a solo esa clase. Se usa un (.) al inicio y se define lo demas en el cuerpo.

  • font-size: Se usa para cambio de tamaño de la letra. Se puede usar XXpx (pixeles), XXrem (tamaño default por el browser) o XXem (tamaño relativo segun el tamaño de la pantalla)

  • background-image: Se usa para colocar una imagen de fondo, teniendo en cuenta que se aplicara a donde sea solicitado. Tiene una forma especifica de declararse: background-image: url(rutaImagen.png);

  • #id: Usado para identificar elementos, preferiblemente y por buenas practicas, un UNICO elemento. Funciona como el caso de las clases. Se debe tener en cuenta que este sobre escribe a los demas.
